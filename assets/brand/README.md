| 파일 | 용도 |
|---|---|
| `banner.png` | README 히어로 배너 (2560×800, 2x) |

(`scripts/check_assets_separation.py`)가 `assets/` 밖의 이미지 바이너리를
차단하기 때문입니다 (CC BY-SA 글리프의 코드 트리 유출 방지 규칙).
